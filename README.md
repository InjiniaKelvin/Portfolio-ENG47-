# Kelvin Mwania Portfolio

Premium personal portfolio built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- Modern dark-theme UI with glassmorphism and animated gradients
- Fully responsive layout with smooth scrolling
- Data-driven content architecture
- SEO-ready metadata and Open Graph assets
- Reusable UI components and motion utilities

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Production

```bash
npm run build
npm run start
```
